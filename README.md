# About the Assignment
This assgingment gives us the task of building a barebones webpage using HTML with the following sections:
- Header with full name and course you're studying
- Short Bio/About Me section
- Why You Are Studying Web Design section
- Career Goals section
- Footer containing name, semester, id, course of study, and year

# Technologies Used
- HTML
- CSS(inline)
- Git
- Github
